CCW

CULTURAL CODE WORD : a word which has no real English translation, because it represents a state of mind unique to Japan. It’s useful to learn these words because they teach you about the Japanese mentality. I’m not trying to teach you all of the cultural code words – there’s plenty of books out there already. But when it comes up, it comes up.
